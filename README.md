# 🚗 Tesla Social Media Analytics for Strategic Branding

## 📌 Overview
This project analyzes Tesla’s social media performance across platforms to generate actionable insights for improving digital marketing strategies.

A complete analytics solution was built using Excel to evaluate engagement, campaign effectiveness, content performance, and ROI from advertising spend.

## 🎯 Objectives
- Identify high-performing content types and hashtags  
- Analyze engagement trends across platforms  
- Evaluate campaign effectiveness and ROI  
- Track follower growth and audience behavior  
- Provide data-driven marketing recommendations  

## 📂 Dataset
The analysis is based on 3 datasets:
- **Posts Dataset** – Post-level engagement metrics (likes, shares, comments, impressions)
- **Engagement Dataset** – Weekly platform performance (followers, ad spend)
- **Campaign Dataset** – Campaign-level metrics (impressions, engagement uplift)

## 🛠 Tools & Skills Used
- Microsoft Excel  
- Data Cleaning & Preprocessing  
- Pivot Tables & Dashboarding  
- Data Visualization (Bar, Line, Scatter plots)  
- KPI Metrics (Engagement Rate, ROI)  

## ⚙️ Process

### 1. Data Cleaning
- Removed duplicates  
- Standardized formats  
- Processed hashtags  

### 2. Engagement Analysis
- Calculated engagement rate  
- Identified top-performing posts  
- Built pivot tables for insights  

### 3. Platform Analysis
- Compared engagement vs ad spend  
- Evaluated platform-wise performance  

### 4. Content Strategy
- Compared videos, images, carousels  
- Identified top-performing hashtags  

### 5. Campaign Analysis
- Measured campaign ROI  
- Analyzed engagement uplift  

### 6. Growth Analysis
- Tracked follower trends  
- Applied moving averages  
- Correlation between ad spend & growth  

## 📊 Key Insights
- 📹 Video content drives the highest engagement  
- 📈 Campaigns significantly boost follower growth  
- 🔍 Not all ad spend translates into engagement → optimization required  
- 🏷️ Specific hashtags improve reach and interaction  

## 🚀 Business Recommendations
- Focus on video-first content strategy  
- Optimize ad spend allocation across platforms  
- Leverage high-performing hashtags consistently  
- Align campaigns with audience engagement patterns
